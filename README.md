# benfords-sawzall

Interactive Python library loaded with constants for real world number crunching

## Setup

It's easier to use this package with [`uv`](https://docs.astral.sh/uv/).

```bash
uv venv
source .venv/bin/activate
```

### Teardown

```bash
deactivate
```

## Usage

```bash
uv run main.py
```
